 <strong><h2><p>Instruções para uso da API</br></br></strong></p>
 
 <p>
1 - Baixe a versão mais recente do Node.js e instale: https://nodejs.org/en/download/ ;</br>

2 - Clone o repositorio (dev ou main) para seu computador;</br>

3 - Abra o terminal e acesse o diretório do repositório em seu computador;</br>

4 - Utilize o comando abaixo para instalar dependências do projeto. <code>npm install</code></br>

5 - Utilize o comando abaixo para instalar o ts-node-dev <code>npm install ts-node-dev</code></br>
Pode ser necessário utilizar o comando abaixo para instalar o ts-node globalmente.</br>
<code>npm install ts-node -g</code></br>

6 -  Acesse a pasta backend/src/ pelo terminal;</br>

7 - Utilize o comando abaixo <code>ts-node.cmd server.ts</code></br> Deve aparecer "Server is Running" no terminal.
</p>
