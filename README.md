<h2 id="descrição"><p><strong>Descrição</strong></br></h2>

<strong style="color:#DE8A0D">Fazendo o Bem</strong> é uma aplicação web criada com o intuito de ajudar pais e alunos de escolas estaduais e municipais à alcançar seus sonhos através do estudo.</br>
Desenvolvido pela equipe do Squad 36 para o Programa de <strong>Formação da FCamara 2021</strong>.</p>


<h2 id="#instalar">
<p><strong>Como Instalar</h2></strong>

• Baixe a versão mais recente do Node.js e siga os passos de instalação: https://nodejs.org/en/download/</br>

• Clone o repositorio main para seu computador</br>

• Abra o terminal e acesse o diretório squad36 em seu computador</br>

• Utilize o comando abaixo para instalar dependências do projeto<br><code>npm install</code></br>

• Utilize o comando abaixo para instalar o ts-node-dev<br><code>npm install ts-node-dev -g</code></br>
</p>

<h2 id="#utilizar">
<p><strong>Como Utilizar</h2></strong>

• Utilize o comando abaixo em seu terminal para ativar a API<br><code>ts-node server.ts</code></br>

• Em caso de sucesso, "Server is Running" aparecerá no terminal<br>

• Acesse o endereço <link>http://localhost:3333</link> em seu navegador<br>
</p>
<p><h3><strong>Funções</strong><br>
• Clique em "Quero Doar" para acessar a página de doação<br>
• Clique em "Preciso de Material" para acessar a página de cadastro de alunos<br>
• Clique em "Cadastrar Escola" para acessar a página de cadastro de instituições de ensino</h3></p>
